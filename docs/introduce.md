# LIGHT CHASER (追光)

> 专注于数据可视化的开源工具

<br/>

# 概述

light chaser 是一款开源免费的数据可视化设计工具，基于 React 技术栈实现。 通过它，你可以简单快速地制作数据可视化相关内容。它可用于大屏幕数据 可视化展示、数据报告、数据分析等场景。
你可以在[github](https://github.com/xiaopujun/light-chaser)上了解它的最新动态：

# 特性

- 丰富的图表类型
- 丰富的图表配置项
- 丰富的快捷键
- 极高的扩展能力
- 内置蓝图编辑器
- 自定义主题
